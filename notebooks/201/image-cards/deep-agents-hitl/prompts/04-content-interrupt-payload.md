竖版 3:4 手绘教育信息图（sketch-notes 卡通手绘风格），与系列同一手绘风格、同一马卡龙配色。

【整体风格】手绘抖动线条，暖奶油纸张背景，马卡龙彩色圆角块，珊瑚红强调；填色留边；手写体中文，深炭灰字。

【标题】顶部手写大标题：「中断数据 · __interrupt__ 里有啥」。

【主体内容】
中央画一个手绘的信封/包裹被拆开，一条弯箭头从「invoke 正常返回」指向它；在包裹旁用珊瑚红醒目标注一句：「暂停 ≠ 结束（invoke 是正常返回的）」。

包裹里飞出两张手绘卡片并排：
左卡（淡蓝）「action_requests · Agent 想干什么」：
  name = write_file
  args = {path:/test.md, content:"Hello World"}
右卡（薄荷绿）「review_configs · 你能怎么回应」：
  allowed_decisions = [approve, reject]

底部手写关键点小字：「拿到这包信息 = Agent 卡在 write_file 前等你拍板 ✋」。

【装饰】角落小星星、问号、对勾。不要渲染颜色名称作为可见文字。
