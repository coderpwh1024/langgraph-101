竖版 3:4 手绘教育信息图封面（sketch-notes 卡通手绘风格）。

【整体风格】手绘抖动线条，所有边框/形状都有轻微手抖感；暖奶油色纸张质感背景 (#F5F0E8)，带细微纸纹；马卡龙柔和配色（淡蓝 #A8D8EA、薰衣草紫 #D5C6E0、薄荷绿 #B5E5CF、蜜桃橙 #F8D5C4），珊瑚红 (#E8655A) 仅用于重点强调；色块填色故意不填满到边缘，保留手绘留白感；无渐变无光泽。文字全部为手写体（中文），深炭灰 #2C3E50。

【画面主体】画面中央一个可爱的卡通机器人（代表 AI Agent），双手抱着一个发光的大脑💭（代表长期记忆）；机器人右侧并排画三个手绘小柜子，从左到右依次贴便利贴、档案柜抽屉、云朵图标（代表三种存储去向）；机器人与柜子之间画一个手绘记忆齿轮/存档桶图标，用波浪箭头连起来。四周散布手绘小星星、波浪箭头、对勾等涂鸦装饰。

【标题文字】顶部大号手写标题：「给 AI Agent 装上长期记忆」；副标题小字：「一张图看懂 Deep Agents 的 Backend 机制」。底部一个圆角徽章标签写「干货收藏 📌」。

【布局】sparse 稀疏布局，主体突出、大量留白，封面感强。不要渲染颜色名称或英文色值作为可见文字。
