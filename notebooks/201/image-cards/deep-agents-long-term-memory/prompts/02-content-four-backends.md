【整体风格】手绘抖动线条，暖奶油色纸张背景 (#F5F0E8) 带纸纹；马卡龙配色彩色圆角卡片（淡蓝/薰衣草紫/薄荷绿/蜜桃橙交替），珊瑚红仅强调；填色不填满边缘；手写体中文，深炭灰文字。

【标题】顶部手写大标题：「四种 Backend · 文件存哪它说了算」，下方一条珊瑚红手绘下划线。

【主体内容】dense 高密度布局，2 行 × 2 列 = 正好 4 个圆角彩色大卡片（不多不少），每个卡片一个手绘类比图标 + Backend 名称 + 类比 + 一句话生命周期：
1. StateBackend —— 便利贴（贴在本场会议白板）—— 线程结束就擦掉（临时）（便利贴图标）
2. FilesystemBackend —— 真实档案柜 —— 存磁盘，重启仍在（永久）（档案柜抽屉图标）
3. StoreBackend —— 公司云盘 —— 跨所有线程持久保留（云朵图标）
4. CompositeBackend —— 前台总台 —— 按路径前缀把文件转交给上面某个柜子（前台铃铛图标）

【底部一句话】手写小字：「Agent 只管说写到哪个路径，存哪由 Backend 决定」。
