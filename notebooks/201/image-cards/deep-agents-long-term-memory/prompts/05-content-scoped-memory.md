竖版 3:4 手绘教育信息图（sketch-notes 卡通手绘风格），与封面同一手绘风格、同一配色，保持系列一致。

【整体风格】手绘抖动线条，暖奶油色纸张背景 (#F5F0E8) 带纸纹；马卡龙配色，珊瑚红仅强调；填色不填满边缘；手写体中文，深炭灰文字。

【标题】顶部手写大标题：「私有 vs 共享 · 隔离开关在哪」，下方一条珊瑚红手绘下划线。

【主体内容】flow 布局，分上下两块：

上半部分「按人隔离」：左右两个卡通小人，左边 Alice、右边 Bob，各自面前一个带锁🔒的私有抽屉，中间画一道手绘隔离虚线并标「互不可见」：
- Alice 抽屉标签：("user", "alice", …)
- Bob 抽屉标签：("user", "bob", …)
中间下方再画一个不带锁、两人手都能伸到的共享抽屉，标签：("shared", …) 标注「全员共享」。

下半部分「隔离开关」：一张手绘代码便签，用珊瑚红高亮一行英文代码。
【重要·英文必须逐字符准确，禁止拼错】这行代码严格显示为：
namespace = lambda rt: ("user", user_id)
其中 lambda 必须拼作 l-a-m-b-d-a（不是 lamnda）；等号必须是 = （不是 > 或 ->）。
下方配一句中文手写说明：「namespace 里动态拼进 user_id → 谁调用就落进谁的桶；shared 写死 → 全员共享」。

【装饰】角落点缀手绘小星星、锁、对勾。不要渲染颜色名称或色值作为可见文字。
